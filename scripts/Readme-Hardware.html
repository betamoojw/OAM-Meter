Geräteinfos

Hier wird aufgeführt, welche Geräte aktuell unterstützt werden. Zusätzlich können verschiedene Firmware-Varianten verfügbar sein, um unterschiedliche Hardwarekonfigurationen auf denselben Geräten zu ermöglichen. Beispielsweise verfügt das REG1 über eine Onboard-Schnittstelle, die wahlweise als zweifacher Binäreingang oder als SML-Eingang genutzt werden kann. Die Spalten zeigen dabei, welcher Kanal in der ETS welcher Schnittstelle zugeordnet wurde.
